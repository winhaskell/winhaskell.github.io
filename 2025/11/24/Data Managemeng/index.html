Course Description

This course provides a comprehensive understanding of modern data management, emphasizing enterprise data architecture, metadata, data quality, governance, security, and the full data lifecycle. Students learn concepts beyond technical database operations, including organizational, ethical, and strategic aspects of data.

Week 1-2: Introduction to Data Management

Why data management matters

Overview of database ecosystem

Relational data model

PostgreSQL / MySQL / SQLite

Week 3-4: Data and Enterprise

The data landscape

Conceptual, logical, physical data models

The common problems with data

manageing data is a business issue

Case study：Banking / Healthcare architecture

Lab：Design a conceptual data architecture for a sample organization

Week 5-6: Data Modeling Foundations

Business rules and constraints

ER modeling

Normalization vs de-normalization

Common enterprise modeling pitfalls

Homework：Model an enterprise domain

Week 7: 中期考试周

考察前三模块内容：搜索与逻辑推理

Week 8: 不确定性与概率推理

贝叶斯网络、马尔可夫过程基础

实例练习：简单概率推理任务

Week 9-10: 机器学习导论

监督学习基础（线性回归、逻辑回归）

无监督学习（K‑means、聚类）

作业：实现分类 & 聚类算法

Week 11: 人工神经网络基础

感知机、多层感知器与反向传播

Week 12: 自然语言处理简介

词袋模型、语言模型基础
