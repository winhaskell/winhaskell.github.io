# 用户使用指南

欢迎来到"绝摆炒饭哥"！本指南将帮助您快速了解如何使用我们的网站。

## 🏠 网站导航

### 主菜单栏（顶部）
- **首页**：浏览我们的招牌美食和品牌特色
- **菜单**：查看所有炒饭种类和价格
- **在线点餐**：添加商品到购物车并下单
- **预订**：提前预订座位
- **关于我们**：了解品牌故事和理念
- **联系我们**：查看地址、电话和营业时间

## 🍚 浏览菜单

1. 点击导航栏的"菜单"或向下滚动
2. 使用分类按钮筛选：
   - **全部菜品**：显示所有16道炒饭
   - **招牌系列**：必点经典款
   - **海鲜系列**：鲜香美味
   - **肉类系列**：香浓可口
   - **素菜系列**：清淡健康
3. 查看每道菜的图片、描述和价格
4. 注意标签：🏆招牌 | 🔥热卖 | ✨新品

## 🛒 在线点餐

### 添加商品
1. 浏览菜单，找到想吃的炒饭
2. 点击"➕ 加入购物车"按钮
3. 系统会显示"已添加到购物车"提示
4. 购物车图标会显示商品数量

### 查看购物车
1. 点击右上角的购物车图标 🛒
2. 弹出购物车窗口
3. 查看已选商品、数量和价格

### 调整订单
- **增加数量**：点击 ➕ 按钮
- **减少数量**：点击 ➖ 按钮
- **删除商品**：点击垃圾桶图标 🗑️
- **清空购物车**：点击"清空"按钮

### 提交订单
1. 确认商品和总价无误
2. 点击"✓ 提交订单"按钮
3. 等待订单确认
4. 看到成功提示即完成

💡 **小贴士**：购物车数据会自动保存，即使关闭浏览器也不会丢失！

## 📅 在线预订

### 预订流程
1. 滚动到"在线预订"部分（或点击导航栏"预订"）
2. 填写必填信息：
   - **姓名**：您的真实姓名
   - **电话**：11位手机号码
   - **日期**：选择就餐日期（只能选择明天及以后）
   - **时间**：选择时间段
     - 午市：11:00-13:00
     - 晚市：17:00-20:00
   - **人数**：选择就餐人数
3. 添加备注（可选）：
   - 靠窗座位
   - 婴儿座椅
   - 庆祝活动
   - 其他特殊需求
4. 点击"✓ 确认预订"按钮
5. 等待预订成功提示

### 预订说明
- ⏰ 建议提前1天预订
- 📞 我们会电话联系确认
- ⭐ 请准时到达，保留时间15分钟
- 🎉 10人以上请提前3天预订
