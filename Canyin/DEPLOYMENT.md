# 部署指南

## 快速部署

要将"绝摆哥炒饭"网站发布上线，请按照以下步骤操作：

### 方法一：使用发布标签页（推荐）

1. 点击页面顶部的 **"Publish"（发布）** 标签页
2. 点击 **"发布"** 按钮
3. 等待发布完成
4. 获取您的网站在线链接

这是最简单、最快速的部署方式！

### 方法二：手动部署到其他平台

如果您想部署到其他平台，可以下载所有文件并上传到：

#### 静态网站托管平台
- **Vercel**：https://vercel.com
- **Netlify**：https://netlify.com
- **GitHub Pages**：https://pages.github.com
- **Cloudflare Pages**：https://pages.cloudflare.com

#### 部署步骤
1. 下载所有项目文件
2. 确保保持以下文件结构：
   ```
   项目根目录/
   ├── index.html
   ├── README.md
   ├── css/
   │   └── style.css
   └── js/
       ├── menu-data.js
       └── main.js
   ```
3. 将整个文件夹上传到托管平台
4. 配置根目录为 `index.html`

## 注意事项

### 生产环境优化建议

当前网站使用 Tailwind CSS CDN 版本，在控制台中会看到警告信息。这在开发阶段没有问题，但如果要在生产环境使用，建议：

1. **保持当前方案**（最简单）
   - 网站功能完全正常
   - 性能足够日常使用
   - 适合中小型项目

2. **升级到生产版本**（可选）
   - 安装 Tailwind CSS 作为 PostCSS 插件
   - 使用 Tailwind CLI 构建
   - 详见：https://tailwindcss.com/docs/installation

### 数据库配置

如果需要启用真实的数据存储（预订和订单）：

1. 数据表已经配置好（`reservations` 和 `orders`）
2. JavaScript 代码中已预留数据库接口
3. 只需确保 RESTful Table API 正常工作即可自动保存数据

### 域名绑定

发布后，您可以：
1. 使用平台提供的免费子域名
2. 绑定自己的自定义域名
3. 配置 HTTPS（大多数平台自动提供）
