## 🍚 项目简介

"绝摆哥炒饭"是一个现代化的餐饮店网站，专注于提供优质的炒饭美食。网站采用时尚潮流与温馨家常相结合的设计风格，提供在线菜单展示、外卖订购、在线预订等完整功能。

### 设计理念
- **时尚潮流**：现代化的UI设计，流畅的动画效果
- **温馨家常**：暖色调配色方案，亲切的用户体验
- **用户至上**：直观的导航，便捷的点餐流程

## 📊 数据存储

### 数据表结构

#### 1. Reservations Table (预订表)
| 字段 | 类型 | 描述 |
|------|------|------|
| id | text | 预订ID |
| name | text | 预订人姓名 |
| phone | text | 预订人电话 |
| date | text | 预订日期 |
| time | text | 预订时间 |
| guests | text | 就餐人数 |
| notes | text | 备注信息 |
| status | text | 状态 (pending/confirmed/cancelled/completed) |
| createdAt | datetime | 创建时间 |

#### 2. Orders Table (订单表)
| 字段 | 类型 | 描述 |
|------|------|------|
| id | text | 订单ID |
| items | array | 订单项目列表 |
| total | number | 订单总金额 |
| orderTime | datetime | 下单时间 |
| status | text | 状态 (pending/confirmed/preparing/ready/delivered/cancelled) |
| customerName | text | 顾客姓名（可选）|
| customerPhone | text | 顾客电话（可选）|
| deliveryAddress | text | 配送地址（可选）|
| notes | text | 订单备注 |

### 本地存储
- **购物车数据**：使用 localStorage 存储，键名：`juebaige-cart`

## 🚀 快速开始

### 在线访问
直接打开 `index.html` 文件即可在浏览器中查看网站。

### 本地开发
1. 下载或克隆项目文件
2. 使用浏览器打开 `index.html`
3. 或使用本地服务器（推荐）：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx serve
   ```

## 📱 功能使用指南

### 在线点餐流程
1. 浏览菜单，选择心仪的炒饭
2. 点击"加入购物车"按钮
3. 点击右上角购物车图标查看订单
4. 调整商品数量或删除不需要的商品
5. 点击"提交订单"完成下单

### 在线预订流程
1. 滚动到"在线预订"部分
2. 填写姓名、电话、日期、时间、人数
3. 添加特殊需求备注（可选）
4. 点击"确认预订"提交

### 查看菜单
1. 点击导航栏"菜单"或滚动到菜单部分
2. 使用分类按钮筛选菜品
3. 查看菜品图片、描述和价格
4. 注意"招牌"、"热卖"、"新品"标签
